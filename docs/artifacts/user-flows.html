User-Flow Wireframes

How the work actually moves

The key journeys through Bricklayer, rebased to Direction v2 — a familiar HubSpot chassis of Dashboards and Reports, with two chat surfaces layered on top: a right-docked, summonable co-working assistant and a full-page threaded chat. Six flows, from first login through building a whole dashboard by conversation to shipping a credit-ready packet.

How to read this  — Each flow is a left-to-right sequence of numbered step-panels joined by arrows; the sequence wraps on narrow screens. These are wireframes of the journey, not pixel UIs — schematics are deliberately rough, but every number is Ed's real POC data ($6.5B book · 3,935 appraisals). This is Voltagent-DS-styled discovery documentation, not the shipping app.
